# Требования к МедКартотеке

## Функциональные требования
- Регистрация и авторизация пациентов и врачей
- Запись пациента к врачу
- Просмотр расписания врачей
- Ведение электронной карты пациента
- Выдача электронных рецептов
- Просмотр результатов анализов

## Нефункциональные требования
- Защита персональных данных (ФЗ-152)
- Быстрый поиск по картотеке
- Доступ через браузер и мобильное приложение
